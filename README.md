# DEESEYEN — финальный продукт (статический 3D-конструктор мебели)

Один файл **`index.html`**: каталог готовой мебели (Three.js), вкладки **«Мебель / Детали»** (куб, цилиндр, столешница, стекло — по мотивам [Flatma](https://flatma.com/create/designer)), **шаблоны** сцены (кухня / шкаф / гостиная), **параметрический корпус** с переключателем **см / мм**, полки, ДСП, **оценка нагрузки**, ручки, **проверка пересечений** (AABB), справка с отсылкой к [PlanPlace](https://planplace.online/), экспорт/импорт **JSON** v1.1, PMREM **RoomEnvironment**.

## Живая версия

https://adilmombekov.github.io/DEESEYEN/

## Локальный запуск

Нужен локальный HTTP-сервер (модуль `three` загружается через importmap с CDN).

```bash
cd DEESEYEN
python -m http.server 8080
```

Открой http://localhost:8080/

## Пересборка `index.html` из эталона

Источник правды в репозитории-папке проекта: `нужное/furnicraft-demo.html`.  
Сборка копирует его сюда и подставляет бренд **DEESEYEN** + мета-теги:

```bash
cd DEESEYEN
python build.py
```
