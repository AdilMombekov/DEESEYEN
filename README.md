# DEESEYEN — финальный продукт (статический 3D-конструктор мебели)

Один файл **`index.html`**: каталог готовой мебели (Three.js), **параметрический корпус** (ширина / высота / глубина в **см**, полки, толщина ДСП), **цвет** (палитра, picker, **RGB**), материалы (дерево, ткань, металл…), **оценка нагрузки** полки (эвристика), **ручки** вкл/выкл, сцена, экспорт/импорт **JSON**, PMREM **RoomEnvironment** для более «дорогого» освещения.

## Живая версия

https://adilmombekov.github.io/DEESEYEN/

## Локальный запуск

Нужен локальный HTTP-сервер (модуль `three` загружается через importmap с CDN).

```bash
cd DEESEYEN
python -m http.server 8080
```

Открой http://localhost:8080/

## Пересборка `index.html` из эталона

Источник правды в репозитории-папке проекта: `нужное/furnicraft-demo.html`.  
Сборка копирует его сюда и подставляет бренд **DEESEYEN** + мета-теги:

```bash
cd DEESEYEN
python build.py
```
